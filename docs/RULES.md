# RULES – Aturan Bisnis Absensi

## 1. Absensi Umum

- Check-in dan check-out wajib waktu
- Validasi lokasi berlaku sesuai kategori
- Foto bersifat kondisional

## 2. Validasi per Kategori

### REGULER

- Toleransi telat: 10 menit
- Lewat → telat per menit

### MAHASISWA

- Toleransi dasar: 10 menit
- Tambahan toleransi perjalanan (tetap)
- Lewat total toleransi → telat per menit

### KEBUN

Wajib:

- waktu
- lokasi
- foto masuk
- foto pulang

## 3. Koreksi Absensi

- Karyawan boleh mengajukan koreksi
- ADMIN boleh koreksi langsung
- Wajib:
  - alasan
  - timestamp
  - before/after value
- Tidak boleh jika payroll period locked

## 4. Lembur

- Diajukan sebelum pulang
- Harus disetujui OWNER
- Dicatat terpisah dari absensi reguler

## 5. Payroll Period

- Unlocked → boleh koreksi
- Locked → read-only
- Lock/unlock hanya oleh OWNER
